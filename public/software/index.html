R Packages

palaeoverse

‘palaeoverse’ is a community-driven software package of generic functions for the palaeobiology community. It provides a standardised toolkit of functions to carry out routine tasks in palaeobiological studies. The package offers a range of functionality including time, latitudinal, and spatial binning, palaeorotation of fossil occurrences, time series plotting, and much more. The package is freely available and has three core principles: (1) streamline data analyses; (2) enhance code readability; and (3) improve reproducibility of results. It is our hope that over time, palaeoverse will be adopted and developed by the community, with contributors and users from all over the world. The package is still currently in development, but its initial release is scheduled for Autumn 2022.

An important note: while I initiated this project, this has been by no means a solo task. I have many amazing collaborators to thank for their contributions. Please do check them out under the contributors section of the GitHub repository

pclimR

‘pclimR’ is an R Package developed by Kilian Eichenseer and I to provide easily-accessible palaeoclimatic data from a range of sources (e.g. δ18O data, palaeoclimatic simulations). In addition, the package provides functionality for implementing a Bayesian approach to reconstruct global mean temperatures and latitudinal temperature gradients.

london

‘london’ is an R Package developed by Paloma Sánchez Osuna and I to provide colour palettes inspired by the City of London. The package includes colours palettes sampled from works housed in Tate Britain, London’s tube map and much more. This R package also provides the option for users to generate their own colour palette from their favourite works or images.
